🏨 Hotel Reservation Analysis (Tableau)

📌 Project Overview

This project analyzes hotel reservation data using Tableau to uncover key insights into booking patterns, customer behavior, and revenue trends. The dashboard provides an interactive view of hotel performance to support better decision-making.

 🎯 Objective

To analyze reservation data and identify patterns, trends, and key factors affecting hotel bookings and overall business performance.

 🛠️ Tools & Technologies

* Tableau
* Excel / CSV Dataset
* Data Visualization Techniques

 📊 Key Features

* Interactive Tableau dashboard with filters and parameters
* Booking analysis by room type, customer segment, and season
* Revenue and occupancy trend visualization
* Identification of peak booking periods
* Customer behavior and cancellation analysis

🔍 Key Insights

* Certain seasons show higher booking rates
* Specific room types are more popular among customers
* Cancellation rates impact overall revenue
* Booking patterns vary across customer segments

 📁 Project Files

* Tableau Workbook (`.twb / .twbx`)
